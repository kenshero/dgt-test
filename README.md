### Installation

install with npm or yarn
```
cd client yarn or npm install
cd compute1/ yarn or npm install
cd compute2/ yarn or npm install
```

# Usage

### First
cd compute1/
```
terminal$ node server.js
```

cd compute2/
```
terminal$ node server.js
```

### Second
cd loadbalancer/
```
terminal$ node index.js
```

### Third
cd client/
```
terminal$ node client.js
```

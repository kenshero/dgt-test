### Installation

install with npm or yarn
```
terminal$ cd client/
yarn or npm install

terminal$ cd compute1/
yarn or npm install

terminal$ cd compute2/
yarn or npm install
```

# Usage

### First
```
terminal$ cd compute1/
terminal$ node server.js
```

```
terminal$ cd compute2/
terminal$ node server.js
```

### Second
```
terminal$ cd loadbalancer/
terminal$ node index.js
```

### Third
```
terminal$ cd client/
terminal$ node client.js
```
